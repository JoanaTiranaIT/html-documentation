HTML Links

<a>

The <a> tag is used to create hyperlinks in HTML. It allows linking to other pages or resources.

Example

<a href="https://example.com">Visit Example</a>

<link>

The <link> tag specifies the relationship between a document and an external resource, often used to link to stylesheets.

Example

<link rel="stylesheet" href="styles.css">

<nav>

The <nav> tag is used to group navigation links, creating a clear structure for navigating a webpage.

Example

<nav>
    <ul>
        <li><a href="#home">Home</a></li>
        <li><a href="#about">About</a></li>
        <li><a href="#contact">Contact</a></li>
    </ul>
</nav>
