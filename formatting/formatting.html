HTML Formatting Tags

<abbr>

The <abbr> tags are used to specify abbreviations or acronyms. You can provide a description via the title attribute.

Example

<abbr title="Hypertext Markup Language">HTML</abbr>

<address>

The <address> tag displays contact information for the author.

Example

<address>
    John Doe

    1234 Elm Street

    Anytown, USA

    john.doe@example.com
</address>

<b>

The <b> tag makes text bold.

Example

<b>Bold text</b>

<bdi>

The <bdi> tag isolates a portion of text that may have a different direction from the surrounding text.

Example

<bdi>Arabic text: مرحبا</bdi>

<bdo>

The <bdo> tag changes the direction of text.

Example

<bdo dir="rtl">Right-to-left text</bdo>

<big>

The <big> tag changes the size of text (not supported in HTML5; use CSS instead).

Example

<big>Larger text</big>

<blockquote>

The <blockquote> tag defines a section that is quoted from another source.

Example

<blockquote>
    <p>This is a blockquote.</p>
</blockquote>

<center>

The <center> tag centers text (not supported in HTML5; use CSS instead).

Example

<center>Centered text</center>

<cite>

The <cite> tag defines the title of a work, such as a book or film.

Example

<cite>The Great Gatsby</cite>

<code>

The <code> tag defines a piece of computer code.

Example

<code>const x = 10;</code>

<del>

The <del> tag indicates deleted text, usually displayed with a line through it.

Example

<del>Deleted text</del>

<dfn>

The <dfn> tag specifies a term that is being defined.

Example

<dfn>HTML</dfn> (Hypertext Markup Language)

<em>

The <em> tag emphasizes text, usually displayed in italics.

Example

<em>Important text</em>

<font>

The <font> tag specifies font, color, and size (not supported in HTML5; use CSS instead).

Example

<font color="red">Red text</font>

<i>

The <i> tag defines text in an italic style.

Example

<i>Italic text</i>

<ins>

The <ins> tag indicates inserted text, usually displayed with an underline.

Example

<ins>Inserted text</ins>

<kbd>

The <kbd> tag represents user input, such as keyboard commands.

Example

<kbd>Ctrl+S</kbd>

<mark>

The <mark> tag highlights text, usually with a background color.

Example

<mark>Highlighted text</mark>

<meter>

The <meter> tag displays a scalar measurement within a known range.

Example

<meter value="0.6">60%</meter>

<pre>

The <pre> tag displays preformatted text, preserving whitespace and line breaks.

Example

<pre>
    Preformatted text
    with spaces and line breaks
</pre>

<progress>

The <progress> tag displays the progress of a task.

Example

<progress value="70" max="100">70%</progress>

<q>

The <q> tag defines a short inline quotation.

Example

<q>This is a quote</q>

<rp>

The <rp> tag defines what to show in browsers that do not support ruby annotations.

Example

<ruby>漢字<rp>(</rp><rt>かんじ</rt><rp>)</rp></ruby>

<rt>

The <rt> tag defines the pronunciation of characters in a ruby annotation.

Example

<ruby>漢字<rt>かんじ</rt></ruby>

<ruby>

The <ruby> tag defines a ruby annotation, often used in East Asian typography.

Example

<ruby>漢字<rt>かんじ</rt></ruby>

<s>

The <s> tag defines text that is no longer correct or relevant, usually displayed with a line through it.

Example

<s>Strikethrough text</s>

<samp>

The <samp> tag defines sample output from a computer program.

Example

<samp>Error: File not found</samp>

<small>

The <small> tag displays text in a smaller size.

Example

<small>Smaller text</small>

<strike>

The <strike> tag defines text that has been struck through (not supported in HTML5; use CSS instead).

Example

<strike>Strikethrough text</strike>

<strong>

The <strong> tag defines important text, usually displayed in bold.

Example

<strong>Important text</strong>

<sub>

The <sub> tag defines subscript text.

Example

<sub>Subscript</sub>

<sup>

The <sup> tag defines superscript text.

Example

<sup>Superscript</sup>

<template>

The <template> tag holds content that is not rendered when the page loads but can be instantiated using JavaScript.

Example

<template>
    <p>This is a template</p>
</template>

<time>

The <time> tag represents a specific period in time.

Example

<time datetime="2024-08-08">August 8, 2024</time>

<u>

The <u> tag defines text with a stylistic underline (use CSS for modern styling).

Example

<u>Underlined text</u>

<var>

The <var> tag defines a variable in a mathematical expression or computer program.

Example

<var>x</var> = <var>y</var> + 2

<wbr>

The <wbr> tag suggests an optional line break opportunity.

Example

Line with longword<wbr>that might break<wbr>here
