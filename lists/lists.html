HTML Lists

<menu>

The <menu> tag is used to create a list of menu items, which is an alternative to the standard <ul> lists.

Example

<menu>
    <li>Item 1</li>
    <li>Item 2</li>
</menu>

<ul>

The <ul> tag creates an unordered list where each list item is marked with a bullet or symbol.

Example

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>

The <ol> tag creates an ordered list where each list item is numbered or lettered.

Example

<ol>
    <li>First item</li>
    <li>Second item</li>
</ol>

<li>

The <li> tag defines an item in a list, whether it is ordered (<ol>) or unordered (<ul>).

Example

<ul>
    <li>List item</li>
</ul>

<dl>

The <dl> tag creates a description list, where each item is defined by a term and its description.

Example

<dl>
    <dt>Term</dt>
    <dd>Description</dd>
</dl>

<dt>

The <dt> tag defines a term in a description list.

Example

<dl>
    <dt>Term</dt>
</dl>

<dd>

The <dd> tag provides the description for a term in a description list.

Example

<dl>
    <dt>Term</dt>
    <dd>Description</dd>
</dl>
