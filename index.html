Introduction to HTML

HTML (Hypertext Markup Language) is the standard language used to create and design web pages. It defines the structure and layout of a web document by using a variety of tags and attributes.

Document Structure

HTML documents are structured with a series of elements that describe the content and layout of the page.

  • <!DOCTYPE html>: Declares the document type.
  • <html>: The root element of the HTML document.
  • <head>: Contains meta-information about the document.
  • <body>: Contains the content of the document.
Text Formatting

HTML provides various tags to format text.

  • <p>: Defines a paragraph.
  • <h1> to <h6>: Define headings of different levels.
  • <strong>: Defines bold text.
  • <em>: Defines italic text.
Media Elements

HTML supports embedding various media elements.

  • <img>: Embeds an image.
  • <audio>: Embeds audio content.
  • <video>: Embeds video content.
  • <iframe>: Embeds another HTML document within the current document.
Forms

HTML forms are used to collect user input.

  • <form>: The container element for form controls.
  • <input>: Defines an input control.
  • <textarea>: Defines a multiline text input control.
  • <button>: Defines a clickable button.
  • <select>: Defines a dropdown list.
