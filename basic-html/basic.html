Basic HTML Tags

<title>

Tag-u <title> tag is used to set the title of an HTML document, which is displayed in the browser's title bar and search engine results.

Shembull

<!DOCTYPE html>
<html>
<head>
  <title>HTML Elements Reference</title>
</head>
<body>

<h1>This is a heading</h1>
<p>This is a paragraph.</p>

</body>
</html>

<body>

Tag-u <body> tag is used to define the main content of an HTML document, such as text, images, and other elements displayed on the web page.

Shembull

<html>
<head>
  <title>Title of the document</title>
</head>

<body>
  <h1>This is a heading</h1>
  <p>This is a paragraph.</p>
</body>

</html>

<h1> to <h6>

Tag-u <h1> to <h6> përdoren për të përcaktuar nivelet e ndryshme të titujve në një dokument HTML, ku <h1> është titulli më i rëndësishëm dhe më i madh, ndërsa <h6> është titulli më i vogël dhe më pak i rëndësishëm.

Shembull

<h1>This is heading 1</h1>
<h2>This is heading 2</h2>
<h3>This is heading 3</h3>
<h4>This is heading 4</h4>
<h5>This is heading 5</h5>
<h6>This is heading 6</h6>

<p>

Tag-u <p> përdoret për të krijuar një paragraf në një dokument HTML.

Shembull

<p>This is a paragraph.</p>

<br>

Tag-u <br> përdoret për të futur një ndarje të vetme rreshti në një dokument HTML, duke krijuar hapësirë vertikale në mes të përmbajtjes.

Shembull

<p>This is the first line.<br>This is the second line.</p>

<hr>

Tag-u <hr> përdoret për të shënuar një ndarje horizontale ku shpesh shfaqet si një vijë horizontale.

Shembull

<hr>

<!-- ... -->

Tag-u <!-- ... --> përdoret për të shtuar komente në një dokument HTML.

Shembull

<!-- This is a comment -->
