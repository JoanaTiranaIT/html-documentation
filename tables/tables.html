HTML Tables

<table>

The <table> tag is used to create a table in HTML.

Example

<table>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

<caption>

The <caption> tag adds a title to the table.

Example

<table>
    <caption>Table Title</caption>
    <tr>
        <th>Header 1</th>
        <th>Header 2</th>
    </tr>
    <tr>
        <td>Data 1</td>
        <td>Data 2</td>
    </tr>
</table>

<th>

The <th> tag defines a header cell in a table.

Example

<table>
    <tr>
        <th>Header</th>
    </tr>
</table>

<tr>

The <tr> tag defines a row in a table.

Example

<table>
    <tr>
        <td>Row data</td>
    </tr>
</table>

<td>

The <td> tag defines a cell in a table.

Example

<table>
    <tr>
        <td>Cell data</td>
    </tr>
</table>

<thead>

The <thead> tag groups the header content in a table.

Example

<table>
    <thead>
        <tr>
            <th>Header 1</th>
            <th>Header 2</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data 1</td>
            <td>Data 2</td>
        </tr>
    </tbody>
</table>

<tbody>

The <tbody> tag groups the body content in a table.

Example

<table>
    <thead>
        <tr>
            <th>Header</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data</td>
        </tr>
    </tbody>
</table>

<tfoot>

The <tfoot> tag groups the footer content in a table.

Example

<table>
    <thead>
        <tr>
            <th>Header</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Data</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>Footer</td>
        </tr>
    </tfoot>
</table>

<col>

The <col> tag specifies column properties.

Example

<table>
    <colgroup>
        <col style="background-color: #f4f4f4;">
        <col>
    </colgroup>
    <tr>
        <td>Data</td>
        <td>Data</td>
    </tr>
</table>

<colgroup>

The <colgroup> tag groups one or more columns in a table for formatting.

Example

<table>
    <colgroup>
        <col style="background-color: #f4f4f4;">
        <col>
    </colgroup>
    <tr>
        <td>Data</td>
        <td>Data</td>
    </tr>
</table>

colspan

The colspan attribute allows a cell to span multiple columns.

Example

<table>
    <tr>
        <td colspan="2">Spans two columns</td>
    </tr>
</table>

rowspan

The rowspan attribute allows a cell to span multiple rows.

Example

<table>
    <tr>
        <td rowspan="2">Spans two rows</td>
        <td>Data</td>
    </tr>
    <tr>
        <td>More data</td>
    </tr>
</table>
