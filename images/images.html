HTML Image Tags

<img>

The <img> tag is used to embed an image in an HTML page.

Example

<img src="image.jpg" alt="Description">

<map>

The <map> tag is used to create an image map, allowing different parts of an image to act as clickable links.

Example

<map name="mapname">
    <area shape="rect" coords="34,44,270,350" href="link.html">
</map>

<area>

The <area> tag is used to define clickable areas within an image map.

Example

<area shape="rect" coords="34,44,270,350" href="link.html">

<canvas>

The <canvas> tag is used to create a drawable region on a page where graphics can be drawn using JavaScript.

Example

<canvas id="myCanvas" width="500" height="400"></canvas>

<figcaption>

The <figcaption> tag is used to provide a caption or description for a <figure> element.

Example

<figure>
    <img src="image.jpg" alt="Description">
    <figcaption>This is an image caption.</figcaption>
</figure>

<figure>

The <figure> tag is used to encapsulate content that is referenced from the main content, like images, diagrams, or videos.

Example

<figure>
    <img src="image.jpg" alt="Description">
    <figcaption>This is an image caption.</figcaption>
</figure>

<picture>

The <picture> tag is used to specify multiple image sources for different conditions, such as screen sizes or resolutions.

Example

<picture>
    <source srcset="image-800w.jpg" media="(min-width: 800px)">
    <source srcset="image-400w.jpg" media="(max-width: 799px)">
    <img src="image-default.jpg" alt="Description">
</picture>

<svg>

The <svg> tag is used to embed Scalable Vector Graphics (SVG) within an HTML page.

Example

<svg width="100" height="100">
    <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>
