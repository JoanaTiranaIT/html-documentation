HTML Styles and Semantics

<style>

The <style> tag is used to include CSS directly within the HTML document.

Example

<style>
    body {
        background-color: #f4f4f4;
    }
</style>

<div>

The <div> tag is a general-purpose container used to group and organize different sections or blocks within a webpage.

Example

<div>
    <p>This is a paragraph inside a div element.</p>
</div>

<span>

The <span> tag is used to group and style small portions of content within an HTML document.

Example

<span style="color: red;">This text is red.</span>

<header>

The <header> tag is used to define a header section in a document or a specific section of a webpage.

Example

<header>
    <h1>Welcome to My Website</h1>
</header>

<hgroup>

The <hgroup> tag groups together headings and subheadings of a section.

Example

<hgroup>
    <h1>Main Title</h1>
    <h2>Sub Title</h2>
</hgroup>

<footer>

The <footer> tag defines a footer for a document or section.

Example

<footer>
    <p>Â© 2024 My Website</p>
</footer>

<main>

The <main> tag identifies and includes the main content of a webpage.

Example

<main>
    <p>This is the main content of the webpage.</p>
</main>

<section>

The <section> tag is used to divide a document into distinct sections.

Example

<section>
    <h2>Section Title</h2>
    <p>Content of the section.</p>
</section>

<search>

The <search> tag is not a standardized HTML tag. Instead, use <form>, <input type="search">, and <button> to create search sections.

Example

<form>
    <input type="search" placeholder="Search...">
    <button type="submit">Search</button>
</form>

<article>

The <article> tag represents a self-contained unit of content that can be distributed or understood on its own.

Example

<article>
    <h2>Article Title</h2>
    <p>This is an article.</p>
</article>

<aside>

The <aside> tag defines content that is outside the main content of the page.

Example

<aside>
    <p>This is an aside section.</p>
</aside>

<details>

The <details> tag is used to show or hide additional information that users can choose to view.

Example

<details>
    <summary>More info</summary>
    <p>Here is some more information.</p>
</details>

<dialog>

The <dialog> tag defines a dialog box or popup window.

Example

<dialog>
    <p>This is a dialog box.</p>
    <button>Close</button>
</dialog>

<summary>

The <summary> tag provides a visible heading for the <details> element.

Example

<details>
    <summary>Details</summary>
    <p>Detailed information goes here.</p>
</details>

<data>

The <data> tag is used to add a machine-readable translation for a given piece of content.

Example

<time datetime="2024-08-08">August 8, 2024</time>
