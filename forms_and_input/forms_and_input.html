HTML Forms and Input Tags

<form>

The <form> tag is used to create an HTML form for user input.

Example

<form action="/submit_form" method="post">
    <input type="text" name="username">
    <input type="submit" value="Submit">
</form>

<input>

The <input> tag specifies an input field where the user can enter data.

Example

<input type="text" name="username">

<textarea>

The <textarea> tag defines a multi-line text input control.

Example

<textarea name="message" rows="4" cols="50">Enter text here...</textarea>

<button>

The <button> tag creates a clickable button.

Example

<button type="button">Click Me</button>

<select>

The <select> tag creates a drop-down list.

Example

<select name="options">
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
</select>

<optgroup>

The <optgroup> tag groups options within a drop-down list created with <select>.

Example

<select name="options">
    <optgroup label="Group 1">
        <option value="1">Option 1</option>
        <option value="2">Option 2</option>
    </optgroup>
</select>

<option>

The <option> tag defines an option in a drop-down list.

Example

<option value="1">Option 1</option>

<label>

The <label> tag defines a label for an <input>, <select>, or other form elements.

Example

<label for="username">Username:</label>
<input type="text" id="username" name="username">

<fieldset>

The <fieldset> tag groups related elements within a form and is often used with <legend> for a description.

Example

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</fieldset>

<legend>

The <legend> tag provides a caption or title for a group of elements within a <fieldset>.

Example

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</fieldset>

<datalist>

The <datalist> tag provides a list of predefined options for an <input> element.

Example

<input list="browsers">
<datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
</datalist>

<output>

The <output> tag displays the result of a calculation or user action.

Example

<output for="num1 num2" name="result"></output>
