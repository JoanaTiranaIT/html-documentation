HTML Forms and Input Tags

<form>

Tag-u <form> përdoret për të krijuar një formë në HTML që lejon përdoruesit të futin të dhëna dhe të dërgojnë ato në një server për përpunim.

Shembull

<form action="/submit_form" method="post">
    <input type="text" name="username">
    <input type="submit" value="Submit">
</form>

<input>

Tag-u <input> përdoret për të krijuar një element të kontrollit të hyrjes në një formë HTML, që lejon përdoruesit të futin të dhëna.

Shembull

<input type="text" name="username">

<textarea>

Tag-u <textarea> përdoret për të krijuar një fushë hyrjeje shumë-rreshtore në një formë HTML, ku përdoruesit mund të futin tekst të gjatë.

Shembull

<textarea name="message" rows="4" cols="50">Enter text here...</textarea>

<button>

Tag-u <button> përdoret për të krijuar një buton në HTML që përdoruesit mund të klikojnë për të ekzekutuar një veprim.

Shembull

<button type="button">Click Me</button>

<select>

Tag-u <select> përdoret për të krijuar një drop-down në HTML, ku përdoruesit mund të zgjedhin një nga opsionet e disponueshme.

Shembull

<select name="options">
    <option value="1">Option 1</option>
    <option value="2">Option 2</option>
</select>

<optgroup>

Tag-u <optgroup> përdoret për të grupuar opsione brenda një drop-down të krijuar me tagun <select> dhe ndihmon në organizimin e opsioneve në kategori më të strukturuara.

Shembull

<select name="options">
    <optgroup label="Group 1">
        <option value="1">Option 1</option>
        <option value="2">Option 2</option>
    </optgroup>
</select>

<option>

Tag-u <option> përdoret për të krijuar një element të veçantë brenda një drop-down list të krijuar me tagun <select>

Shembull

<option value="1">Option 1</option>

<label>

Tag-u <label> përdoret për të krijuar një etikëtë që ofron një përshkrim ose titull për një element të formës në HTML, si input-et, butonat e radios, ose checkbox.

Shembull

<label for="username">Username:</label>
<input type="text" id="username" name="username">

<fieldset>

Tag-u <fieldset> përdoret për të grupuar dhe organizuar elementët brenda një forme në HTML dhe zakonisht shoqërohet me tagun <legend> për të ofruar një përshkrim të grupit.

Shembull

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</fieldset>

<legend>

Tag-u <legend> përdoret për të ofruar një përshkrim ose titull për një grup elementësh që grupon tagu <fieldset>.

Shembull

<fieldset>
    <legend>Personal Information</legend>
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
</fieldset>

<datalist>

Tag-u <datalist> përdoret për të ofruar një listë opsionesh që janë të paracaktuara dhe që mund të shfaqen si sugjerime për përdoruesit në fusha të hyrjes<input> element.

Shembull

<input list="browsers">
<datalist id="browsers">
    <option value="Chrome">
    <option value="Firefox">
    <option value="Safari">
</datalist>

<output>

Tag-u <output> përdoret për të shfaqur rezultatin e një llogaritjeje ose të dhënave që janë prodhuar nga një veprim i përdoruesit në një formë HTML.

Shembull

<output for="num1 num2" name="result"></output>
